Angular Google Plus Sign-in Directive

Usage

  1. Include google-plus-signin.js.
  2. Add directive.g+signin as a dependency to your app.
  3. Add <g+signin clientid="your-client-id">s.
  4. Create a signinCallback() function to detect successful authentication. Sample in the README.

<div ng-app="directive.g+signin">
